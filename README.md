# UFO
Drive an UFO in FiveM.

<img src=https://imgur.com/vexru3M.png>

## Installation
* Download the resource ;
* Drag and drop it in your resources folder ;
* Add ```start ufo``` to you ```server.cfg```.
G = no wheel
E = no stationary mode
